# Garmin Health Analysis (OpenClaw Skill)

This repository contains the **OpenClaw** skill for querying Garmin Connect health/activity metrics and generating interactive HTML charts.

- Primary documentation: **[SKILL.md](SKILL.md)**

## Install

```bash
# via clawhub
clawhub install garmin-health-analysis

# or manually
mkdir -p ~/.openclaw/skills
cd ~/.openclaw/skills
git clone https://github.com/eversonl/ClawdBot-garmin-health-analysis.git garmin-health-analysis
```

Project links:
- OpenClaw: https://openclaw.ai
- ClawHub: https://clawhub.com
